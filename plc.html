주제 및 목표

프로젝트 목표

MPS 장비를 이용하여 소재 자동 분류 및 적재·저장 시스템을 구현하고, 서보모터 제어 기술을 적용하여 자동화 공정의 효율성을 향상시키는 것을 목표로 한다.

또한 HMI를 구축하여 장비 상태를 실시간으로 확인할 수 있도록 하였으며, 소재 판별, 위치 제어, 공정 모니터링 기능을 통합적으로 구현하였다.

금속 및 비금속 소재를 자동으로 판별하고 지정된 위치로 적재 및 배출하는 자동화 시스템을 구축하여 작업 효율성과 생산성을 향상시켰다.

핵심 공정

  1. 초기 점검 공정

장비 전원 인가 후 전체 실린더 후진 , 원점 복귀 및 장치 동작 여부를 확인하는 공정이다.

  1. 소재 판별 공정

센서를 이용하여 소재를 금속과 비금속으로 자동 판별하며, 판별 결과에 따라 이송 경로를 자동으로 설정한다.금속 소재는 4개 이상 누적 시 배출 실린더를 통해 외부로 배출하며, 비금속 소재는 컨베이어를 통해 저장 박스로 자동 이송된다.

  1. 자리 비움 공정

총 6개의 소재가 창고에 적재된 후 사용하는 공정, 전체 소재를 1층부터 3층까지 순차적으로 컨베이어를 통해 이동시키고 저장 박스로 자동 배출한다.

  1. 선택 적재 및 비움 기능

사용자가 원하는 층과 위치를 지정하여 소재를 적재하거나 비울 수 있는 기능을 제공한다.

핵심 알고리즘 설명

소재 적재 및 비움 제어 알고리즘

본 시스템은 선택 적재·선택 비움 기능과 전체 비움 기능을 하나의 위치 제어 구조로 통합하여 설계하였다.

동일한 서보 위치 이동 로직을 공유하면서도, 기능 간 충돌을 방지하기 위해 명확한 동작 구분과 인터락 구조를 적용하였다.

  1. 선택 적재 및 선택 비움 로직

선택 적재 및 선택 비움 기능은 위치 버튼별 메모리 기반 방식으로 구현하였다.

사용자가 특정 층과 위치를 선택하면, 해당 버튼에 대응되는 메모리에 저장된 ZP(서보 위치 데이터)를 호출하여 서보모터가 지정된 위치로 이동하도록 구성하였다.

이 방식은 티칭 기능을 통해 각 층의 위치 값을 유연하게 수정할 수 있으며,

위치 변경 시 프로그램 수정 없이 대응 가능하도록 하여 유지보수성과 확장성을 고려한 설계이다.

  1. 전체 비움 로직

전체 비움 기능은 창고에 적재된 모든 소재를 사전에 정의된 순서에 따라 자동으로 배출하도록 설계하였다.

배출 순서는 금속 소재를 1층부터 3층까지 먼저 배출한 후, 비금속 소재를 1층부터 3층까지 배출하는 구조이다.

해당 로직은 카운터 기반 순차 제어 방식을 사용하여 총 6단계로 구성하였다.

  • 카운트 0 : 금속 1층
  • 카운트 1 : 금속 2층
  • 카운트 2 : 금속 3층
  • 카운트 3 : 비금속 1층
  • 카운트 4 : 비금속 2층
  • 카운트 5 : 비금속 3층

카운트 값에 따라 서보 위치를 자동으로 전환하여, 전체 소재가 누락 없이 순차적으로 배출되도록 구현하였다.

  1. 기능 구분 및 충돌 방지 인터락 설계

선택 비움 기능과 전체 비움 기능은 동일한 위치 이동 알고리즘을 공유하기 때문에,

동시에 동작할 경우 오동작이 발생할 가능성이 있다.

이를 방지하기 위해 선택 비움 전용 메모리를 B접점 방식으로 구성하였다.

선택 비움 동작 시에는 해당 접점이 OFF 상태가 되어 전체 비움 로직이 동작하지 않도록 차단되며,

이를 통해 두 기능을 명확히 구분하고 동시 동작에 의한 충돌을 원천적으로 방지하였다.

이와 같은 인터락 구조를 통해 시스템의 동작 안정성, 신뢰성, 유지보수성을 향상시켰다.

서보모터 구성과 설명

1. 서보 시스템 구성

본 설비는 정밀 위치 제어를 위해 서보 모터 제어 방식을 적용하였다.
서보 제어 구성은 다음과 같다.

PLC → 위치결정 모듈 → Servo Amp → Servo Motor → Servo Amp(Feedback)

PLC에서 위치 명령을 생성하고, 위치결정 모듈을 통해 서보 앰프로 전달한다.
서보 앰프는 명령 값에 따라 서보 모터를 구동하며, 엔코더 피드백을 통해 위치와 속도를 제어한다.

  1. 문제 상황

동작 완료 시간을 측정하도록 회로를 구성하였으나, 공정 종료 이후에도 시간이 누적되어 실제 동작 시간과 다른 값이 표시되는 문제가 발생하였다.

  1. 원인 분석

  1. 해결 방법

공정 시작 조건을 추가하여 동작 시작 시 시간 측정 레지스터가 초기화되도록 회로를 수정하였다.

결과

각 공정 수행 시 실제 동작 시간만 측정되도록 개선되었으며, 공정 시간 모니터링의 신뢰성을 확보할 수 있었다.

  1. 선택 적재 인터락

선택 적재 동작 이후에는 일반 자동 적재가 수행되지 않도록 구성하였다.

선택 적재로 창고 위치가 변경된 상태에서 자동 적재가 수행될 경우, 적재 위치 오류가 발생할 수 있기 때문이다.

따라서 선택 적재 이후 적재 작업은 선택 적재 모드에서만 가능하도록 설계하였다.

선택 배출 수행 후에는 자동 적재 동작을 제한하였다.

  1. 선택 배출 인터락

선택 배출로 인해 창고 내부 소재 배치가 변경되므로, 자동 적재 수행 시 소재 관리 오류가 발생할 수 있다.

이후 적재 작업은 선택 적재를 통해 수행하도록 구성하였다.

  1. 선택 적재 시 소재 판별 제외

선택 적재 동작은 사용자가 직접 위치를 지정하는 기능이므로, 자동 운전에서 사용하는 소재 판별 로직을 적용하지 않도록 구성하였다.

이를 통해 사용자가 지정한 위치로 바로 이동하도록 설계하였다.

개선 효과

  1. 택타임 측정 기능 구현

QCPU(Q 모드) / QnACPU 프로그래밍 매뉴얼의 특수 메모리 SM412(1초 클록 신호)를 활용하여 택타임 측정 기능을 구현하였다.
SM412 신호와 INCP 명령어를 사용하여 임의 레지스터 값을 1초 단위로 증가시키도록 구성하였으며, 이를 통해 공정 시작부터 동작 완료까지의 시간을 측정할 수 있도록 하였다. 따라서 작업 시간 확인이 가능해졌으며, 공정 효율 분석 및 작업 조건 개선에 활용할 수 있었다.

  1. 비상 정지 시 안전 동작 개선

비상 정지 발생 시 가공 실린더는 즉시 상승하도록 구성하였으며, 서보 모터는 동작 완료를 기다리지 않고 축 정지 기능을 통해 즉시 정지하도록 설계하였다. 따라서 비상 상황 발생 시 장비 충돌 위험을 감소시키고 작업 안전성을 향상시킬 수 있었다.

One-paper Maintenance Manual

One-paper Maintenance Manual

  1. 설비 개요

본 설비는 투입된 소재를 금속 및 비금속으로 판별하여 자동으로 분류하고, 선택된 창고 위치에 적재·저장 및 배출을 수행하는 PMS 자동화 장비이다.

소재 판별 결과에 따라 금속 소재는 좌측 창고, 비금속 소재는 우측 창고로 이송되며, 각 창고는 1층부터 3층까지 총 6개의 적재 위치로 구성되어 있다.

설비는 서보 모터를 이용한 정밀 위치 제어 방식을 적용하였으며, 설비 기동 시 자동 원점확립(원점복귀)을 수행하여 초기 위치를 설정한다.

HMI를 통해 다음 정보를 실시간으로 확인할 수 있다.

  • 서보 모터 현재 위치 및 속도
  • 서보 동작 상태
  • 경고 번호 및 에러 번호
  • 금속 / 비금속 판별 상태
  • 금속 수량 / 비금속 수량 / 전체 생산 수량
  • 작업 상태 메시지
  • 동작 완료 시간

또한 HMI를 통해 가공 횟수 및 동작 시간 등 공정 설정 값 변경이 가능하다.

본 설비는 자동 운전, 동작 점검, 수동 JOG 조작, 위치 티칭, 개별 적재 기능, 개별 및 전체 창고 비움 기능을 지원한다. HMI 램프 표시를 통해 창고 적재 상태, 설비 동작 가능 상태, 비상 상태, 창고 비움 상태 및 원점 확립 상태를 확인할 수 있다.

  1. 설비 운전 준비

설비 운전 전 다음 조건을 모두 확인해야 한다.

  • 비상정지 스위치 해제 상태
  • PLC READY 램프 점등 상태
  • 서보 모터 원점확립 완료 상태
  • 서보 모터 에러 및 경고 번호 0 상태
  • 설비 비상 동작 상태 아님
  • 동작 가능 램프 점등 상태
  • 동작 점검 기능 수행 시 장비 정상 동작

위 조건을 만족할 경우 설비 운전이 가능하다.

  1. 소재 투입
  • HMI 동작 화면에서 동작 버튼 실행
  • 소재 미투입 대기 시간이 0이 되면 동작 완료
  1. 수동 조작 방법

4.1 JOG 운전

서보 화면에서 JOG 버튼을 이용하여 위치 미세 조정 수행

4.2 위치 티칭

  • JOG 운전으로 목표 위치 이동
  • 티칭 버튼 입력 후 위치 저장

4.3 적재 및 비움 기능

개별 적재 : 원하는 층 선택 후 적재 수행

개별 비움 : 원하는 층 선택 후 저장 박스로 소재 배출

창고 비움 : 전체 층 비움 동작 수행

비움 동작 중 비움 중 램프 점등

  1. 이상 발생 시 조치 방법

비상 정지 발생 시

  • 비상정지 해제 후 알람 상태 확인
  • 원점확립 수행 후 운전 재개

서보 알람 발생 시

  • 알람 코드 확인 및 원인 제거
  • 서보 리셋 후 원점복귀 수행

설비 동작 불가 시

  • 동작 가능 램프 및 인터락 조건 확인
  • 필요 시 설비 초기화 또는 원점복귀 수행

적재 및 배출 이상 발생 시

  • 적재 상태 및 티칭 위치 확인
  • 필요 시 JOG 운전으로 위치 보정
