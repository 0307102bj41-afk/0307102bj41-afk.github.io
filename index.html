Project 02

STM32 RC Control

02
Embedded C · STM32 · Real-time Control

STM32 RC Control

조이스틱 입력, 블루투스 통신, LCD UI, PWM 모터 제어를 통합한 레지스터 기반 임베디드 시스템. 입력 수집·통신·출력·예외 처리를 하나의 시스템으로 연결해 실제 펌웨어 구조를 end-to-end로 이해하고 구현했습니다.

유형 학습형 · 코드리뷰형
진행 방식 개인 프로젝트
중심 주제 Embedded C / STM32
구현 범위 Input · Comm · UI · Motor
핵심 구현 포인트
  • ADC 입력 처리 — 2축 조이스틱을 12bit 값(0~4095)으로 변환
  • Custom UART Packet — 방향·버튼·체크섬 기반 커스텀 제어 패킷 설계
  • LCD UI — 1602 LCD + I2C 변환 모듈 상태 표시
  • PWM 모터 제어 — 좌우 독립 제어로 직진·회전·정지 구현
  • Interrupt & Debounce — 버튼 채터링 억제 및 안정적 입력 처리
  • State Machine Parser — 수신 패킷 단계별 파싱으로 통신 신뢰성 확보
기술 키워드
STM32F411 Cortex-M4 Embedded C Register-level ADC UART / BT I2C LCD PWM Motor SysTick Timer Interrupt
실무형 역량으로 해석
Data In
샘플링 시간·클럭 분주를 고려한 아날로그 입력 안정적 수집 구조 이해
Integrity
체크섬·상태 플래그·타임아웃으로 통신 무결성 및 예외 대응 설계
Realtime
SysTick·Timer·Interrupt로 CPU 점유 최소화·주기적 동작 분리
Modular
adc.c · uart.c · i2c.c · motor.c 기능별 모듈 분리 → 리뷰·유지보수 용이
트러블슈팅 & 학습 성과
  • ADC 쓰레기 값변환 완료 플래그 대기 후 읽기로 안정적 입력값 확보
  • I2C 동기화LCD 초기화 시퀀스 조정으로 통신 안정성 개선
  • UART 예외타임아웃·예외 핸들러 도입으로 무한 대기 방지
  • 문서화저수준 레지스터 제어를 설명 가능한 구조로 정리, 코드리뷰 포트폴리오 소재로 전환
