STM32 RC Control

02

STM32 RC Control

조이스틱 입력, 블루투스 통신, LCD UI, PWM 모터 제어를 통합한 레지스터 기반 임베디드 시스템. 입력 수집·통신·출력·예외 처리를 하나의 시스템으로 연결해 실제 펌웨어 구조를 end-to-end로 구현했습니다.

유형학습형 · 코드리뷰형
진행 방식개인 프로젝트
중심 주제Embedded C / STM32
구현 범위Input · Comm · UI · Motor
핵심 구현 포인트
  • ADC 입력 처리 — 2축 조이스틱을 12bit 값(0~4095)으로 변환
  • Custom UART Packet — 방향·버튼·체크섬 기반 커스텀 제어 패킷 설계
  • LCD UI — 1602 LCD + I2C 변환 모듈 상태 표시
  • PWM 모터 제어 — 좌우 독립 제어로 직진·회전·정지 구현
  • Interrupt & Debounce — 버튼 채터링 억제 및 안정적 입력 처리
  • State Machine Parser — 수신 패킷 단계별 파싱으로 통신 신뢰성 확보

Project 03

PLC MPS 자동화 시스템

03
PLC · Servo Motor · HMI · Industrial Automation

MPS 기반 서보 정밀 위치 제어 & 소재 자동 분류 저장 시스템

PLC(Q03UDV)와 서보 모터를 활용해 금속·비금속 소재를 자동 판별하고, 3층 창고에 정밀 위치 제어로 적재·배출하는 산업 자동화 시스템. HMI 구축을 통해 실시간 모니터링과 다국어 지원을 구현했습니다.

유형팀 미니 프로젝트
CPUMitsubishi Q03UDV
소프트웨어GX Works2 / GT Designer3
구현 범위PLC · Servo · HMI 통합
핵심 공정
  • 초기 점검 공정 — 전원 인가 후 전체 실린더 후진 및 원점 복귀 확인
  • 소재 판별 공정 — 센서 기반 금속/비금속 자동 판별 후 이송 경로 설정. 금속 4개 이상 누적 시 배출, 비금속은 컨베이어로 저장
  • 자리 비움 공정 — 창고 6개 소재 전체를 1~3층 순차 컨베이어 배출
  • 선택 적재/비움 — 사용자가 층·위치 지정하여 개별 적재 또는 배출
시스템 아키텍처
PC / HMI (GT Designer3)
PLC Q03UDV — 위치결정 모듈 QD77MS2
Servo Amplifier → Servo Motor
엔코더 Feedback → 위치·속도 제어
소재 판별 → 적재/배출 실린더 제어
핵심 구현 기능
  • 동적 가중치 서보 위치 제어 — ABS/INC 방식, AXIS Pointer INC로 층간 높이만 설정해 설정값 최소화
  • 카운터 기반 순차 배출 — 6단계 카운터(금속 1~3층 → 비금속 1~3층) 전체 비움 로직
  • B접점 인터락 설계 — 선택 비움↔전체 비움 충돌 방지, Fail-Safe 이중 안전 구조
  • HMI 실시간 모니터링 — 서보 위치/속도/에러, 소재 수량, 작업 상태 메시지
  • 택타임 측정 — SM412 1초 클록 + INCP 명령으로 공정 시간 측정
  • 다국어 지원 — Word 스위치 + Comment 기능으로 한국어/영어 전환
기술 스택
Mitsubishi PLC Q03UDV GX Works2 GT Designer3 Servo Motor QD77MS2 위치결정 CC-Link AD/DA 변환모듈 Ladder 프로그래밍 HMI GOT2000 Fail-Safe 인터락

안전 설계

  • HW Limit (FLS/RLS) — B접점(NC) 물리 스위치, 단선·전원차단 시 자동 정지하는 Fail-Safe 구조
  • SW Limit Logic — PLC 가상 위치 벽으로 HW 리미트 도달 전 이중 차단
  • 선택 적재 인터락 — 선택 적재 후 자동 적재 차단, 적재 위치 오류 방지
  • 비상 정지 — 즉시 서보 축 정지 + 가공 실린더 상승, 충돌 위험 제거

트러블슈팅

  • 타임 누적 문제 공정 종료 후에도 시간이 누적되는 버그 → 공정 시작 시 레지스터 초기화 조건 추가로 해결
  • 서보 위치 충돌 선택 비움·전체 비움 동시 동작 오류 → B접점 인터락으로 기능 간 충돌 원천 차단
  • INC 최적화 층별 절대 위치 개별 저장 → AXIS Pointer INC 전환으로 설정값 감소·유지보수성 향상
